==== 空調運転
　空調時間は表5.3.2に示す通りとする。空調稼働日は1F中央監視室を除き、図5.3.10のカレンダーパターンに示す平日（年間247日）とする。
(3)計算対象負荷
建物全体テストでは室負荷のみを評価対象とし、外気負荷は対象外とする。ただし、空調システムシミュレーションツールテスト－全体システムテスト（6.2節）において外気負荷を含めたシステムシミュレーションを行うため、表5.3.2に各室の外気導入量を示している。
(4)壁体の構成
壁体部材は表5.3.3に示す構成とする。各居室・ゾーンに面する外壁の面積・壁体種別は表5.3.4に示す通りとする。各居室間の間仕切の面積・壁体種別は表5.3.5、表5.3.6、表5.3.7に示す通りとする。
(5)天井裏の扱い
事務室の空調システムは天井レターン方式とする。熱負荷計算では天井裏を含めて一室と扱い、天井裏の外壁面積も熱負荷の要素として考慮する。同様に、室の空気の熱容量も天井裏まで含めた室容積で与える。ただし、隙間風および家具の熱容量の算定は、天井裏を含まない室の容積を基準とする。
(6)床下地盤境界の扱い
地下ピットを非空調隣室として設定し、地盤境界は設定しない。
(7)隙間風
隙間風の想定は換気回数法を採用し、全室において換気回数を0.1回/hとする。換気回数を計算する際の容積には天井裏空間は含めない。換気回数は空調時・非空調時で変化しないものとする。
(8)内部発熱
表5.3.2に示す照明発熱、機器発熱、人員密度より、各居室の発熱条件を設定する。照明発熱、機器発熱の放熱の比率は60%が放射、40%が対流とする。照明発熱において、プログラム内部で安定器損失を上乗せして発熱量が計算される場合、その損失分を加味した発熱量が表5.3.2に記載の数値になるように与える。機器発熱において冷却方法は強制空冷とし、100％対流により放熱されるものとする。人体発熱に関しては人の体表面積を1.7m2、作業状態は事務作業（代謝量1.2 met）を想定し、全熱120W/人（顕熱80W/人、潜熱40W/人）とする。各発熱のスケジュールは表5.3.8に示す通りとする。発熱スケジュールの正時間の補間方法は線形補間とする。
(9)不透明部材表面の放射特性
外壁の短波吸収率80％、長波放射率90％とする。室内側の相互反射や相互放射の計算を行う場合には同様の数値で設定する。
(10)室外側熱伝達率
室外側は対流熱伝達率18W/m2K、放射熱伝達率5W/m2K、総合熱伝達率23W/m2Kとする※1。
(11)室内側総合熱伝達率
室内側は対流熱伝達率4W/m2K、放射熱伝達率5W/m2K、総合熱伝達率9W/m2Kとする※1。
(12)窓ガラス
表5.3.4に各居室の窓面積及びブラインドの有無の設定を示す。ガラスは透明フロートガラス8mmとし、ブラインドは中間色とする。窓ガラスは日射の透過率81.5％、反射率7.2％とする。透過率、反射率の入射角特性は「JISA2103附属書G 斜入射に対する板ガラスの光学特性の計算方法」の透明フロート板ガラスを標準とするが、ユーザが特性を入力できないツールの場合にはデフォルト値で計算して良い。ブラインドは奥行き25mm、ピッチ22.5mm、日射反射率は両面とも0.66とし、年間を通してスラット角90°でブラインドは下ろした状態を想定する。
(13)室内日射分配
特に規定しない。
(14)室内の設定温湿度
夏期(6～9月)：26℃，50%
冬期（12～3月）：22℃，40%
中間期（4,5,10,11月）：24℃，50%
(15)計算時間間隔
非定常シミュレーションの離散時間は原則1時間とするが、それより短い間隔で計算してもよい。その場合、結果に1時間ごとの値が求められる場合はシミュレーション結果から1時間（正時の前後30分）の平均値を算出する。
(16)ゾーン間換気量
隣接するゾーン間に内壁が存在しない場合は、ゾーン間空気移動量はゾーン間境界長さ当たり150m3/(h・m)とする。空気移動量は空調時・非空調時で変化しないものとする。
(17)隣室条件
非空調室が隣接している場合、計算対象室と外気の中間温度を仮定する（1F床面は地下部をピットと想定して隣室扱いとし、同様の仮定とする）。計算の簡略化のため、上下階は計算対象室と同じ温度とする。
(18)家具熱容量
家具の熱容量を考えるのは居室のみとし、通過空間は空気分の熱容量しか考えない。なお、通過空間とは、廊下、EVホール、自販機コーナーをいう。
(a)通過空間（廊下等）：室の空気容積と同じ熱容量とする。
(b)居室：室の空気容積の10倍の熱容量とする。
なお、湿気容量は空気分のみ考慮する。
(19)柱・梁の熱容量
表5.3.9をもとに、事務室のみ柱、梁の熱容量（すべて普通コンクリートとする）を考慮する。
(20)特別休日
1月3日および12月29日は特別日として設定し、祝日と同じ扱いとする。（図5.3.10）
(21)空調機の最大熱処理能力
空調機の最大熱処理能力は無限大と仮定し、過負荷状態は想定しない。
(22)初期条件
本計算の前に助走計算（例えば、12月の最終日3日間）を行うこととする。助走計算開始時の初期条件は22℃、40％とする。
(23)空調エリア・非空調エリア
空調エリア・非空調エリアは図5.3.11のように設定する。
(24)指定した計算条件を入力できない場合の取り扱い
熱負荷計算ツールによっては、例えば、総合熱伝達率をユーザが変更できないなど、ガイドラインで指定した計算条件を入力できない場合がある。このような場合は、ガイドラインで指定した計算条件を満足せずとも、ユーザが設定可能な範囲の条件のみを設定してテストを実施し、7章で示したチェックリストに、対応できなかった設定条件が何かと、それらをどのように与えたのかを明記する。
